# Site Photographie - Projet Immersion

Ce projet est conçu pour être un portfolio de photographie haut de gamme, minimaliste et performant.

## Structure du Projet

- `/backend` : API FastAPI pour la gestion des albums et des métadonnées des photos.
- `/frontend` : Application Next.js utilisant Tailwind CSS et Framer Motion pour une interface immersive.
- `product.md` : Guide stratégique, technique et design du projet.
- `image.pdf` : Référence visuelle originale.

## Démarrage Rapide

### Backend
Allez dans le dossier `backend` et lancez le serveur :
```bash
pip install fastapi uvicorn pydantic
python main.py
```

### Frontend
Allez dans le dossier `frontend` et lancez l'application (requiert `npm` ou `yarn`) :
```bash
npm install
npm run dev
```
